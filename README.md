 Password Strength Analyzer

Aplikacja desktopowa w Pythonie (Tkinter) do analizy siły haseł
oraz symulacji popularnych metod łamania haseł.

 Funkcje
- analiza siły hasła
- test schematyczności (human pattern test)
- atak maskowy (mask attack)
- brute-force (symulacja)
- test słownikowy
- pasek postępu i raport

 Pliki
- index.py – główna aplikacja GUI
- human_pattern_test.py – test schematycznych haseł
- mask_test.py – atak maskowy

 Wymagania
- Python 3.10+
- pytest (do testów)

## Uruchomienie
```bash
python index.py
